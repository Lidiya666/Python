Основная цель проекта: на основе данных создать модель, которая может по ключевым параметрам предсказать исход экзамена.

Данные: их много, они достаточно чистые (мало ошибок), пустые значения заполнены, выбросы присутствуют.

Этапы: изучить все данные, проверить их по графикам, исключить ошибки, проверить корреляции, проанализировать.
